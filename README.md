# Tabbed Out: Subverting the Android Custom Tab Security Model

## Artifacts

This repository contains artifacts for the paper "Tabbed Out: Subverting the Android Custom Tab Security Model", submitted to CCS 2023. We provide the source code and results of the Custom Tab analysis in the wild and Proof of Concepts for the proposed attacks.

### Folder Structure

- `analysis` contains code and results for the Custom Tab analysis.
- `pocs` contains Proof of Concepts for the proposed attacks.
